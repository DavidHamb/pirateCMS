Default methods

What do we do when faced with a new, unfamiliar service?

  • Is there a web interface ? See the web service methodologies ... Just like those linked to port 80.

  • Find the version

  • Can't you find default credentials ?

  • Searchsploit and Metasploit results ?

  • Type in Google :
    • "service_name service_version exploit"
    • "service_name AND site:hackingarticles.in"
    • "service_name AND site:hacktricks.xyz"

  • Read the documentation :
    • Identify dependencies : one of them could be vulnerable
    • Identify default credentials
    • Read any chapter dealing with security issues
    • Become familiar with basic usage
    • Get the source code and read it (do some reverse engeneering)

  • Finally, report all usefull findings in a new methodology !
